## Dashboard Overview

The Dashboard is your landing page after logging in. From here, you can:
- View a summary of real-time metrics, recent detection events, and alerts.
- See visual statistics and notification tiles (e.g., number of alerts, active geofences, drone status).
- Access navigation links for Geofences, Detections, Drones, Alerts, and Analytics.

## Geofences

**What are Geofences?**

Geofences are predefined polygons on the map that represent areas of interest (e.g., forests, high-risk zones).

**Using Geofence Features:**
1. Go to the **Geofences** page from the sidebar or dashboard.
2. View existing geofences overlaid on the map.
3. Select a geofence to see its details, associated detection events, and network status.
4. **Create/Edit/Delete**: Users with permissions can add new geofences or modify existing ones by interacting with the map drawing tools.
5. Geofences are linked to network quality/status for rapid incident response.

## Drones

Drones are employed to enhance monitoring within geofenced regions.

- Go to the **Drones** page to:
    - View live drone locations and their network quality.
    - See recent activities or detection events related to each drone.
    - Click on a drone to see its telemetry, status, and any alarms.

## Detection Events & Alerts

**Detection Events:**
- Events are generated automatically when sensors, drones, or other systems detect a bushfire anomaly.
- View detection events from the **Detection Events** or **Alerts** page.

**Alerts:**
- Critical alerts are generated based on detection event types and thresholds.
- Each alert provides:
    - Time of event
    - Geofence or drone location
    - Severity level
    - Associated actions (e.g., SMS sent to subscribers, notifications)

- Click an alert to see detailed info and historical data for the area/device.

## FAQ

**Q: Who receives SMS/email alerts?**
A: Only subscribed users linked to particular geofences or roles.

**Q: Can I add new geofences?**
A: Only users with admin or certain user roles can create or modify geofences.

**Q: Is my data private?**
A: All user and event data is secured. Contact your admin for specifics on privacy and compliance.
